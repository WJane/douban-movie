## 项目描述

* 豆瓣电影，通过豆瓣API爬取电影数据，并且用react完成前端构建。


###### 技术栈
* react
* redux
* react-router


###### 构建
* create-react-app


## 运行
- 克隆远程库
```
git clone https://github.com/WJane/douban-movie.git
```
- 进入项目目录
```
cd douban-movie
```
- 安装依赖
```
npm install
```
- 打包生成
```
npm run build
```
- 开发
```
npm start
```
- 打开浏览器输入http://localhost:3000/
